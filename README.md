# Country catalog

### [Live demo](https://timurash.github.io/country-catalog/#/)

### Description

This project is integration with the [REST Countries API](https://restcountries.com/)

The original goal was to try out the most up-to-date front-end tools
at the moment like composition API, Vite bundler, pNPM package manager and etc

### Technology stack

- [Vue 3 (Composition API)](https://vuejs.org)
- [Vue Router](https://router.vuejs.org)
- [TypeScript](https://www.typescriptlang.org)  
- [Element Plus UI library](https://element-plus.org) 
- [Vite](https://vitejs.dev)
- [Jest](https://jestjs.io)
- [Vue Testing Library](https://testing-library.com)
- [pNPM package manager](https://pnpm.io)
